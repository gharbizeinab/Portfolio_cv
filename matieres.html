Matières Principales

  • Programmation Web : Apprentissage de HTML, CSS, JavaScript et frameworks frontend.
  • Base de Données : Conception et manipulation de bases de données relationnelles avec MySQL.
  • Architecture des Ordinateurs : Analyse et conception des systèmes informatiques.
  • Mathématiques pour l'Ingénieur : Concepts avancés pour la résolution de problèmes techniques.
  • Probabilités et Statistiques : Analyse des données et modélisation statistique.
  • Systèmes d'Exploitation : Fonctionnement et gestion des systèmes informatiques.
  • Transmission des Données : Étude des techniques de communication et des protocoles.
  • Algorithmique et Programmation : Développement en langages C et Python.
  • Systèmes Logiques : Conception et analyse des circuits logiques.
  • Préparation à la Certification DELF : Perfectionnement en langue française.

Projets Réalisés

Projet : Site Personnel

Description : Création d’un site web personnel comprenant plusieurs pages pour présenter des compétences, projets et contacts.

Outils : HTML, CSS, JavaScript.

Projet : Site Web de l’IIT

Description : Réalisation d’un site web pour l’institut IIT basé sur des maquettes fournies.

Outils : HTML, CSS, Media Queries.

Projet : Radar avec Arduino

Description : Conception d’un système radar capable de détecter des objets et de les visualiser à l'aide d'une carte Arduino et d'un capteur ultrason.

Outils : Arduino, capteur ultrason, Processing.

Travaux Pratiques

  • TP 3 : Création et intégration des feuilles de style CSS dans des pages HTML.
  • TP 4 : Utilisation des Media Queries pour rendre un site responsive avec CSS3.
  • TP 5 : Validation d’un formulaire avec JavaScript et manipulation dynamique de contenu.
